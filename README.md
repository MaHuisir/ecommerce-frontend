# LUMINA — 光影艺术

沉浸式摄影作品展示网站

## 技术栈

- **Vue 3** (Composition API)
- **Vite 5** (构建工具)
- **Vue Router 4** (路由)
- **GSAP** (动画)
- **Lenis** (平滑滚动)
- **Sass** (CSS 预处理器)

## 页面

- **Home** — 首页轮播 + 精选作品
- **Gallery** — 作品瀑布流
- **Work Detail** — 作品详情
- **About** — 摄影师介绍

## 本地开发

```bash
npm install
npm run dev
```

## 构建

```bash
npm run build
```
